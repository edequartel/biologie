## 🔗 Escape room voor leerlingen  
Plaats deze link op het digibord of deel met de klas:

👉 [Escape room – De bloedsomloop](https://edequartel.github.io/biologie/openlessen/bloedsomloop.html)

**Belangrijk:**  
Op deze pagina staat ook de **link naar de codekraker**. Nadat leerlingen alle vragen hebben gemaakt en hun 10-cijferige code hebben verzameld, kunnen zij via deze website direct hun code invoeren en controleren of deze goed is.

## Handleiding – Escape Room als vervangende les

Deze escape room is een **volwaardige biologieles** waarbij leerlingen zelfstandig werken en de leerstof herhalen.

### Wanneer gebruik je deze les?
- Bij lesuitval  
- Als herhalingsles  
- Voor of na een toets  
- Als afsluiting van een hoofdstuk  

### Lesopzet (50–60 minuten)

**Start (5 min)**  
Lees voor:  
> “Jullie gaan vandaag een escape room doen. Werk alleen of in tweetallen. Beantwoord de vragen, verzamel de cijfers en kraak daarna jullie code via de link op de website.”

**Escape room (35–45 min)**  
- Leerlingen openen de escape room via de link  
- Beantwoorden de vragen  
- Elke vraag levert één cijfer op  
- Samen vormen deze cijfers een code  
- Via de website openen zij de codekraker en controleren hun code  

**Afronding (5–10 min)**  
Bespreek kort welke vragen lastig waren.

### Voorbereiding (1 minuut)
- Zet de link op het digibord  
- Laat leerlingen de website openen  
- Controleer vooraf of de pagina werkt  

### Belangrijk
De vervangende docent hoeft:
✅ geen uitleg te geven  
✅ geen biologiekennis te hebben  
✅ niets na te kijken  

Alleen helpen bij praktische vragen.

### Werkt iets niet?
Laat leerlingen:
1. Hun antwoorden opnieuw controleren  
2. Nakijken of alle cijfers goed zijn overgenomen  
3. De code opnieuw invoeren via de codekraker op de website  

### Kort samengevat
1. Deel de link naar de escape room  
2. Laat leerlingen alleen of in tweetallen werken  
3. Laat ze via de website hun code kraken  

Meer is niet nodig 🙂
